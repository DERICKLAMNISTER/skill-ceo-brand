# ANÁLISIS DEL VIDEO DE BRANDING VIRAL

## CONCEPTO

**Mensaje Central:** Cómo crear contenido viral en redes sociales mediante tres pilares fundamentales: valor, entretenimiento y emoción.

**Framework SPCL (Smart Brands Create Love):**
- **S**hares (Compartir) - Máxima valoración del algoritmo
- **B**ookmarks (Guardar) - Contenido de referencia
- **C**omments (Comentarios) - Provocar reacción
- **L**ikes (Me gusta) - Mínima valoración algorítmica

**Insight clave:** "La gente no comparte buen contenido, comparte contenido que los hace ver bien"

**Jerarquía de impacto:**
1. 10K vistas = 1 pilar
2. 100K vistas = 2 pilares combinados
3. 1M+ vistas = Los 3 pilares integrados

## MARCOS

### Marco 1: **La Trinidad del Contenido Viral**
- **Valor** (Value): Aprendizaje tangible
- **Entretenimiento** (Entertains): Retención de atención
- **Emoción** (Emotion): Impulso de compartir

### Marco 2: **Jerarquía Algorítmica SPCL**
Orden de importancia según plataformas:
1. Shares (compartir) - 40% peso algorítmico
2. Bookmarks (guardar) - 30%
3. Comments (comentar) - 20%
4. Likes (me gusta) - 10%

### Marco 3: **Ecuación de Viralidad**
```
Viralidad = Valor × Entretenimiento × Emoción
```
- 1 elemento = Alcance básico
- 2 elementos = Alcance medio
- 3 elementos = Potencial viral máximo

### Marco 4: **Filtro de Creación de Contenido**
Antes de publicar preguntarse:
- ¿Por qué les importaría? (relevancia)
- ¿Por qué lo compartirían? (moneda social)

## APLICACIÓN

### Para emprendedores/pequeños negocios:

**Paso 1: Audita tu contenido actual**
```
Analiza tus últimas 10 publicaciones:
- ¿Cuántas tienen valor claro?
- ¿Cuántas entretienen?
- ¿Cuántas generan emoción?
- ¿Cuántas combinan 2+?
```

**Paso 2: Crea matriz de contenido SPCL**
| Tipo de contenido | Valor | Entretiene | Emoción | Potencial |
|-------------------|-------|------------|---------|-----------|
| Tutorial paso a paso | ✓ | | | Bajo |
| Before/After | ✓ | ✓ | ✓ | Alto |
| Behind the scenes | | ✓ | ✓ | Medio |

**Paso 3: Aplica filtros antes de publicar**

**Checklist pre-publicación:**
- [ ] ¿Enseña algo específico y aplicable?
- [ ] ¿Mantiene atención hasta el final?
- [ ] ¿Genera reacción emocional clara?
- [ ] ¿La gente querrá compartirlo para verse bien?
- [ ] ¿Tiene gancho en primeros 3 segundos?

---

### Plantilla de contenido viral:

**Estructura de 60 segundos:**

```
0-3seg: Hook impactante con métrica/resultado
3-10seg: Promesa de lo que aprenderán
10-40seg: Explicación con visual atractivo
40-55seg: Aplicación práctica/call to action
55-60seg: Cierre memorable + CTA seguir
```
