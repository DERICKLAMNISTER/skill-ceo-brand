## CONCEPTO

### Mensaje Central
El video presenta un sistema completo de identidad de marca basado en **5 tipos de logos** y **7 estilos visuales**. El concepto fundamental es que una marca moderna necesita múltiples versiones de su logo para adaptarse a diferentes contextos y aplicaciones.

### Propuesta de Valor
- **Versatilidad**: Un sistema de logos flexible para cualquier situación
- **Profesionalismo**: Estructura completa de identidad corporativa
- **Adaptabilidad**: Soluciones para cada medio y formato
- **Coherencia**: Mantener la identidad en todas las versiones

### Público Objetivo
- Emprendedores y startups
- Dueños de pequeñas y medianas empresas
- Diseñadores gráficos junior
- Marketers digitales
- Profesionales independientes construyendo su marca personal

## ESTRATEGA (Análisis Estratégico)

### Los 5 Tipos de Logos - Función Estratégica

#### 1. **Main Logo (Logo Principal)**
```
Propósito: Identidad primaria
Uso: Papelería, sitio web principal, publicidad grande
Características: Versión completa con todos los elementos
Ejemplo: SportsBuy con ícono y texto completo
```

#### 2. **Secondary Logo (Logo Secundario)**
```
Propósito: Adaptabilidad horizontal/vertical
Uso: Firmas de email, banners, layouts diferentes
Características: Reorganización de elementos
Ejemplo: SportsBuy con ícono arriba y texto abajo
```

#### 3. **Simplified Logo (Logo Simplificado)**
```
Propósito: Legibilidad en espacios pequeños
Uso: Favicons, apps móviles, merchandising pequeño
Características: Solo texto o solo símbolo
Ejemplo: "SportsBuy" sin ícono
```

#### 4. **Submark (Submarca)**
```
Propósito: Identificación compacta
Uso: Redes sociales, iconos, sellos, marcas de agua
Características: Versión cuadrada/circular mínima
Ejemplo: Solo el ícono de 4 cuadrados
```

#### 5. **Animated Logo (Logo Animado)**
```
Propósito: Medios digitales dinámicos
Uso: Videos, GIFs, intro/outro, redes sociales
Características: Movimiento que refuerza identidad
Aplicación: Video intros, stories, reels
```
