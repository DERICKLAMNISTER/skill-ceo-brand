# ANÁLISIS COMPLETO DEL VIDEO DE BRANDING

## CONCEPTO

**Premisa central:** Democratizar el lujo visual a través de paletas de colores estratégicas.

**Propuesta de valor:**
- Enseña que el lujo no requiere presupuestos millonarios, sino decisiones cromáticas inteligentes
- Ofrece fórmulas replicables (3 paletas específicas)
- Promete transformación tangible ("brand makeover")

**Gancho psicológico:**
- Apela al deseo universal de parecer premium/establecido
- Usa el concepto "old money luxury" (aspiracional)
- Genera urgencia con sorteo exclusivo

---

## VISUAL

### **Jerarquía de información:**
1. **Códigos HEX** prominentes (credibilidad técnica)
2. **Nombres descriptivos** de colores (memorabilidad)
3. **Texturas metálicas** fotografiadas (tactilidad)
4. **Imágenes contextuales** (aplicación real)

### **Composición:**
- **Simetría vertical** en slides de paletas (equilibrio = lujo)
- **Fondos oscuros** (#1D1D1F aprox.) para resaltar colores
- **Espacio negativo generoso** (respiración visual)
- **Grid 4x1** en paletas (orden cognitivo)

### **Tratamiento fotográfico:**
- **Iluminación cálida y suave** (invita, no intimida)
- **Profundidad de campo selectiva** (enfoque en productos)
- **Tonos desaturados estratégicamente** (sofisticación)

## ESTRATEGIA

### **Regla 60-30-10 explicada:**
```
60% → Color primario (dominante emocional)
30% → Color secundario (soporte tonal)
10% → Acento metálico (jerarquía visual)
```

**Por qué funciona:**
- Evita sobrecarga cognitiva
- Crea puntos focales predecibles
- Permite consistencia multiplataforma

### **Principios de lujo aplicados:**

1. **Restricción cromática**
   - Máximo 4 colores = disciplina = exclusividad
   
2. **Saturación profunda**
   - Colores "ricos" (#3a1012 vs #ff0000)
   - Evita primarios puros (parecen baratos)

3. **Metales como protagonistas**
   - Gold/Rose-Gold/Copper = valor tangible
   - Texturas fotografiadas (no planos)

4. **Neutrales cálidos**
   - Beige/Crema vs blanco/gris
   - Sensación de materiales naturales

### **Hooks de engagement:**

- **Apertura:** "3 color palettes that make your brand feel luxurious"
  - Número impar (completo pero manejable)
  - Verbo activo "make" (transformación)

- **Cierre:** "Enter website + handle for brand makeover"
  - CTA de doble acción (aumenta compromiso)
  - Incentivo valioso (servicio premium gratis)

## APLICACIÓN (CÓMO USARLO)

### **Para emprendedores/startups:**

**Paso 1: Selección de paleta**
```
Heritage → Si vendes: Joyería, vinos, legal, finanzas tradicionales
Botanical → Si vendes: Wellness, eco-productos, hoteles boutique
Executive → Si vendes: Consultoría, tech B2B, educación premium
```

**Paso 2: Distribución 60-30-10**
```
Ejemplo Heritage:
- 60% Borgoña (#3a1012) → Fondos de secciones clave
- 30% Crema (#dbd0c2) → Fondos generales, tarjetas
- 10% Gold → CTAs, íconos, bordes de elementos premium
- Neutral (#9d8062) → Textos secundarios, footers
```

**Paso 3: Jerarquía tipográfica**
```
H1 → Serif display (The Seasons) en primario
H2 → Sans serif bold (Montserrat) en secundario  
Body → Sans serif regular en neutral oscuro
Accent → Script (Sloop) solo en logo/firmas
```

### **Para diseñadores:**

**Audit de marca existente:**
1. ¿Usas más de 4 colores? → Simplifica
2. ¿Tus colores están 100% saturados? → Baja a 70-85%
3. ¿Faltan metálicos? → Agrega como acento
4. ¿Tu blanco es #FFFFFF? → Cámbialo por crema (#f5
