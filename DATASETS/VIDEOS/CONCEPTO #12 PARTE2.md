## MARCOS (Frameworks Conceptuales)

### 1. **Los 7 Tipos de Logos:**

| Tipo | Definición | Cuándo usar |
|------|------------|-------------|
| **Pictorial Marks** | Logos basados en imagen única | Marca visual fuerte, reconocible |
| **Lettermarks** | Monogramas de iniciales | Nombres largos que necesitan recordación |
| **Wordmarks** | Solo tipografía | Nombre de marca es el activo principal |
| **Combination Marks** | Texto + símbolo | Máxima flexibilidad, más común |
| **Emblems** | Texto dentro de símbolo | Instituciones, tradición, confianza |
| **Abstract Marks** | Símbolos sin significado literal | Crear significado con el tiempo |
| **Mascots** | Personajes representativos | Construir personalidad y conexión emocional |

### 2. **Los 4 Sistemas de Arquitectura de Marca:**

| Sistema | Estructura | Ejemplo | Aplicación |
|---------|------------|---------|------------|
| **Branded House** | Una marca, múltiples servicios | FedEx (Express, Ground, Freight, Office) | Servicios relacionados bajo una marca |
| **Sub-brands** | Una marca, muchos productos | Apple (News, Music, Watch, tv+) | Productos derivados con identidad propia |
| **Endorsed Brands** | Múltiples marcas respaldadas | Marriott (Courtyard, Fairfield, Residence Inn) | Diversidad con respaldo corporativo |
| **House of Brands** | Marcas independientes | P&G (cada producto es autónomo) | Portfolio diversificado sin conexión visible |

## APLICACIÓN

### Para Emprendedores/Startups:
1. **Evaluar tu nombre**: ¿Es largo? → Considera lettermark
2. **Definir tu esencia**: ¿Tradicional? → Emblem. ¿Moderno? → Abstract mark
3. **Planificar crecimiento**: ¿Múltiples servicios? → Branded house desde el inicio

### Para Empresas Establecidas:
1. **Auditar tu sistema actual**: ¿Tu arquitectura refleja tu estrategia?
2. **Consistencia**: ¿Todos tus touchpoints usan el logo correctamente?
3. **Evolución**: ¿Necesitas simplificar para digital? (de combination a pictorial)

### Para Diseñadores:
1. **Justificar decisiones**: Usa este framework en presentaciones a clientes
2. **Expansión del proyecto**: Propón sistemas completos, no solo logos aislados
3. **Caso de estudio**: Titanark muestra cómo implementar sub-brands coherentes

### Checklist de Decisión:

**¿Qué tipo de logo necesitas?**
- [ ] ¿Tu nombre es descriptivo y corto? → **Wordmark**
- [ ] ¿Tienes iniciales memorables? → **Lettermark**
- [ ] ¿Necesitas símbolo reconocible? → **Pictorial**
- [ ] ¿Quieres flexibilidad? → **Combination**
- [ ] ¿Representas tradición/autoridad? → **Emblem**
- [ ] ¿Quieres diferenciarte visualmente? → **Abstract**
- [ ] ¿Tu marca es amigable/divertida? → **Mascot**

**¿Qué arquitectura necesitas?**
- [ ] ¿Un servicio con variantes? → Branded House
- [ ] ¿Productos con identidad propia? → Sub-brands
- [ ] ¿Adquisiciones que mantener? → Endorsed Brands
- [ ] ¿Portfolio diverso sin relación? → House of Brands
