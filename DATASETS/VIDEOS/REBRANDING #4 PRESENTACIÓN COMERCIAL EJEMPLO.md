# Análisis del Video de Branding de Twix

## CONCEPTO

**Idea Central:** Demostración práctica de cómo un mismo producto puede reposicionarse para diferentes segmentos de mercado mediante el rediseño de empaque y messaging.

**Estrategia de Segmentación:**
- 5 audiencias completamente distintas
- Mismo producto base, diferentes propuestas de valor
- Adaptación visual y comunicacional específica para cada nicho

**Objetivo Educativo:** Mostrar que el branding no es solo estética, sino comunicación estratégica de beneficios específicos para cada público objetivo.

---

## VISUAL

### 1. **Twix Active (Fitness)**
- **Colores:** Negro con fucsia/morado eléctrico
- **Estética:** Energética, dinámica, deportiva
- **Elementos:** Rayos eléctricos, tipografía bold
- **Contexto:** Mancuernas, pesas, ambiente gym

### 2. **Twix Raw (Orgánico)**
- **Colores:** Kraft natural, marrón tierra
- **Estética:** Rústica, artesanal, honesta
- **Elementos:** Ilustraciones de cacao, textura papel
- **Mensaje:** "Ethically grown organic cacao"

### 3. **Twix Thin (Dietético)**
- **Colores:** Dorado/beige claro, minimalista
- **Estética:** Limpia, delicada, ligera
- **Elementos:** "125 calories" prominente
- **Formato:** Barra más delgada y alargada

### 4. **Twix Junior (Niños)**
- **Colores:** Pasteles suaves, multicolor
- **Estética:** Alegre, infantil, amigable
- **Elementos:** Sol sonriente, estrella, arcoíris
- **Mensaje:** "100% Real Milk", nutrientes esenciales

### 5. **Twix Reserve (Premium)**
- **Colores:** Negro mate con detalles plateados
- **Estética:** Lujosa, sofisticada, exclusiva
- **Elementos:** Caja texturizada con sleeve
- **Detalles:** "85% cacao", "Ecuadorian dark chocolate"

## MARCOS

### **Estructura Narrativa:**
1. **Hook inicial:** "Watch me take this Twix label..."
2. **Presentación secuencial** de cada variante
3. **Fórmula repetitiva:** Audiencia → Nombre → Características visuales → Beneficios
4. **Cierre con síntesis** del aprendizaje
5. **Call to action:** "Follow for more..."

### **Timing:**
- Ritmo rápido y dinámico
- Transiciones limpias entre productos
- Suficiente tiempo para absorber cada concepto
- Video corto y consumible (formato TikTok/Reels)

## ESTRATEGIA

### **Principios de Segmentación Aplicados:**

1. **Beneficio funcional diferenciado:**
   - Proteína y recuperación (Active)
   - Salud y ética (Raw)
   - Control calórico (Thin)
   - Nutrición infantil (Junior)
   - Experiencia sensorial (Reserve)

2. **Códigos visuales de categoría:**
   - Cada diseño utiliza el lenguaje visual esperado por su audiencia
   - No hay confusión sobre a quién está dirigido

3. **Precio implícito:**
   - Desde económico (Thin) hasta premium (Reserve)
   - El empaque comunica el nivel de precio

4. **Canales de distribución sugeridos:**
   - Active: Tiendas deportivas, gyms
   - Raw: Tiendas orgánicas, health stores
   - Thin: Supermercados, sección dietética
   - Junior: Supermercados, zona infantil
   - Reserve: Tiendas gourmet, duty-free

### **Lección Estratégica:**
> "By taking the same product and giving it five different labels, we can appeal to five completely different audiences."

**Key Insight:** El packaging no solo protege, COMUNICA y POSICIONA.

## APLICACIÓN

### **Para Emprendedores y Marcas:**

#### **1. Antes de Diseñar, Pregúntate:**
- ¿A quién le estoy vendiendo específicamente?
- ¿Qué problema les resuelvo?
- ¿Qué valores son importantes para ellos?
- ¿Qué códigos visuales reconocen y valoran?

#### **2. Ejercicio Práctico:**
Toma tu producto y crea 3 versiones para:
- **Audiencia A:** [Define características demográficas y psicográficas]
- **Audiencia B:** [Diferente edad, valores, necesidades]
- **Audiencia C:** [Distinto poder adquisitivo, estilo de vida]

#### **3. Elementos a Adaptar:**

**a) Naming:**
- Debe comunicar el beneficio clave
- Fácil de entender
- Memorable

**b) Color:**
- Negro/metálicos → Premium, deportivo
- Kraft/naturales → Orgánico, honesto
- Pasteles → Infantil, suave
- Minimalista → Diet, saludable

**c) Tipografía:**
- Bold → Energía, fuerza
- Serif elegante → Lujo, tradición
- Sans-serif limpia → Modernidad, claridad
- Redondeada → Amigable, infantil

**d) Materiales/Formato:**
- Papel kraft → Sostenibilidad
- Mate negro → Premium
- Brillante → Convencional
- Caja vs. wrapper → Nivel de precio

**e) Messaging:**
- Funcional: "8G Protein"
- Emocional: "Guilt-free"
- Ético: "Ethically grown"
- Técnico: "85% cacao"

#### **4. Testing de Concepto:**

Antes de producir, valida:
- ¿La audiencia objetivo reconoce que es "para ellos"?
- ¿El beneficio es claro en 3 segundos?
- ¿Se diferencia de la competencia?
- ¿Es coherente con el precio?
