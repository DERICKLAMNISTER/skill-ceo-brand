# ANÁLISIS COMPLETO DEL REBRANDING DE MAEVE

## CONCEPTO

**Estrategia de arquitectura de marca dual:**
- Mantienen "Seattle Chocolate" como marca paraguas (credibilidad + 30 años de trayectoria)
- Crean "Maeve" como submarca juvenil y creativa
- Eliminan la limitación geográfica sin perder la herencia
- Concepto mitológico: Maeve = reina guerrera irlandesa
- Fusionan: energía femenina + magia del noroeste pacífico + chocolate ético

**Posicionamiento:**
- De chocolate regional → experiencia mítico-sensorial global
- Target: consumidor millennial/Gen Z que valora storytelling + ética + diseño
- Propuesta: cada sabor es un universo narrativo propio

---

## VISUAL

**Sistema de ilustración:**
- Estilo: psicodélico contemporáneo con referencias a art nouveau + cultura pop
- Paleta: colores saturados y vibrantes (rosas, azules, verdes, amarillos)
- Personajes: diosas, astronautas, sirenas, criaturas fantásticas
- Cada sabor tiene identidad visual propia pero coherente con el sistema

**Arquitectura gráfica:**
- Marco consistente estilo "tarot card" o "poster de concierto vintage"
- Tipografía bold para MAEVE (impacto + legibilidad)
- Bordes decorativos geométricos que unifican la línea
- Elementos whimsicos: flores, estrellas, olas, elementos naturales

**Ejecución:**
- Ilustración 100% custom (no stock)
- Máxima "instagrameability" - cada paquete es contenido
- Funciona tanto en físico como en pantallas

## MARCOS (Frameworks conceptuales aplicados)

1. **Modelo de Extensión de Marca (Keller)**
   - Transferencia de equity: Seattle Chocolate → Maeve
   - Nuevo territorio sin abandonar credibilidad

2. **Storytelling de Marca (Campbell - Hero's Journey)**
   - Cada sabor = personaje con historia propia
   - Consumidor como explorador de mundos

3. **Branding Sensorial**
   - Experiencia visual precede experiencia gustativa
   - Empaque como primer punto de contacto emocional

4. **Purpose-Driven Branding**
   - Mujer-owned + éticamente sourced = valores integrados
   - No es marketing, es ADN de marca

5. **Tribal Marketing**
   - Crean comunidad alrededor de coleccionabilidad
   - Fans = evangelizadores orgánicos

## ESTRATEGA (Decisiones estratégicas clave)

### ✅ ACIERTOS MAGISTRALES:

1. **No matar la marca madre**
   - "Seattle Chocolate Presents Maeve" = transición inteligente
   - Mantienen distribución + relaciones comerciales

2. **Crear plataforma infinita**
   - Universo Maeve permite lanzamientos ilimitados
   - Cada sabor nuevo = nuevo personaje + historia

3. **Diseño como producto**
   - El empaque es tan valioso como el chocolate
   - Genera UGC (user-generated content) orgánico

4. **Targeting generacional sin alienar**
   - Atractivo para Gen Z pero no excluye otros segmentos
   - La calidad del chocolate retiene consumidor tradicional

5. **Omnicanalidad nativa**
   - Funciona igual en shelf, DTC, Instagram, TikTok
   - Packaging diseñado para ser fotografiado

### 🎯 ESTRATEGIAS APLICADAS:

- **Differentiation through design** en categoría commoditizada
- **Narrative architecture** (cada SKU cuenta una historia)
- **Community building** vía coleccionabilidad
- **Premium positioning** sin elitismo (accesible pero aspiracional)

## APLICACIÓN (Cómo replicar esta estrategia)

### PARA MARCAS ESTABLECIDAS:

**1. Audita tu equity actual**
```
- ¿Qué es intransferible de tu marca?
- ¿Qué te limita geográfica/conceptualmente?
- ¿Puedes crear una "hija" sin matar la "madre"?
```

**2. Define tu universo expandido**
```
- Si tu marca fuera un mundo, ¿qué mitología tendría?
- ¿Qué arquetipos representan tus productos?
- ¿Cómo se conectan entre sí?
```

**3. Invierte en sistema visual propietario**
```
- No templates
- No stock art
- Contrata ilustrador/estudio con visión
- Crea sistema flexible pero reconocible
```

### PARA NUEVAS MARCAS:

**1. Empieza con el mundo, no solo el producto**
```
- ¿Tu marca es un lugar? ¿Una época? ¿Una filosofía?
- Ejemplo Maeve: mitología celta + PNW + empoderamiento femenino
```

**2. Diseña para viralidad**
```
- Cada elemento debe ser "posteable"
- Piensa en cómo se ve en mano
- Considera el unboxing experience
```

**3. Construye coleccionabilidad**
```
- Sabores limitados
- Packs temáticos
- Colaboraciones con artistas
```
