## APLICACIÓN

### Fórmula Extraíble para Marcas de Lujo

**PASO 1: Color Psychology**
```
Wealth → Verde, oro, marrón, beige
Prosperity → Vino, malva, negro, gris
Success → Azul petróleo, teal, gris
```

**PASO 2: Typography Hierarchy**
```
Headlines → Serif clásica (Griffon)
Subheads → Sans serif espaciada (Aviano)
Accents → Script elegante (Nautica)
Body → Sans moderna limpia (Gravesend)
```

**PASO 3: Brand Archetype Selection**
```
Para lujo/premium → "The Ruler"
↓
Comunicar: Control, poder, estatus
Evitar: Debilidad, fracaso, inseguridad
Demostrar: Superioridad (sin arrogancia)
```

**PASO 4: Coherencia Visual Total**
- Fotografía: Iluminación dramática, composiciones asimétricas
- Lenguaje: Aspiracional pero no pomposo
- Contenido: Valor + exclusividad (no solo venta)
- Experiencia: Concierge approach en todos los puntos de contacto

### Checklist de Implementación

**Para redes sociales (caso Instagram)**:
- [ ] Logo icónico simple (funciona en tamaño pequeño)
- [ ] Color primario del arquetipo elegido
- [ ] Grid cohesivo (90%+ del color primario)
- [ ] Bio con ubicación premium
- [ ] Contenido educativo/aspiracional (no solo producto)
- [ ] Servicio visible ("Concierge 24/7")
- [ ] Fotografía profesional consistente
- [ ] Tipografía personalizada en gráficos

**Para marca general**:
- [ ] Definir arquetipo primario
- [ ] Palette de 3-4 colores alineados
- [ ] Sistema tipográfico de 3-4 fuentes
- [ ] Guía de tono de voz
- [ ] Iconografía/personalidad visual
- [ ] Caso de estudio/aplicación real
