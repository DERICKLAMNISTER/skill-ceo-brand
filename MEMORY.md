## 🧠 CONOCIMIENTO ESTRUCTURADO

### NIVEL 1: ARQUITECTURA VISUAL

#### Color Theory
- **Psicología:** Rojo (energía), Azul (confianza), Verde (naturaleza)
- **Aplicación:** 60-30-10 rule en paletas
- **Premium signaling:** Oro, negro, blanco para lujo

#### Typography
- **Serif:** Autoridad, tradición
- **Sans-serif:** Modernidad, limpieza
- **Script:** Elegancia, feminidad
- **Display:** Personalidad fuerte
- **Regla:** 2-3 typefaces máximo en marca

#### Logo Design
- **Requisitos:** Legible, memorable, versátil, atemporal, diferenciado
- **Tipos:** Wordmark, symbol + wordmark, abstract symbol
- **Scalability:** Debe funcionar de favicon (16px) a billboard (10m)

#### Premium vs Luxury Matrix

- **Luxury:** 10-50x+ base price, exclusivity-driven, ultra-high net worth
- **Entry:** Accessible luxury for aspirational mass

### NIVEL 3: EXPERIENCIA DIGITAL

#### Social Media Strategy
```
Educational (50%) → Provide value
Entertaining (30%) → Drive engagement
Promotional (20%) → Direct sales
```

#### Voice & Tone
- **Voice:** Consistent personality across all channels
- **Tone:** Adapts to context (formal announcement vs casual story)
- **Messaging Pillars:** 3-5 key messages repeated consistently

#### Visual Consistency
- Color palette adherence (brand colors in 80%+ of content)
- Typography selection (approved fonts only)
- Photography style (lifestyle, minimal, raw, filtered, etc.)
- Graphic treatment language (geometric, ornate, minimal, etc.)

---

### NIVEL 4: PSICOLOGÍA DEL CONSUMIDOR

#### Memory Formation
- **Exposure:** Need 7+ touches for top-of-mind
- **Sensory:** Logo + sound + smell + texture unique
- **Emotional:** Stories trigger stronger retention
- **Repetition:** Consistent messaging across touchpoints
- **Association:** Brand linked to emotions/values

#### Decision-Making Triggers
- **Scarcity:** Limited availability (FOMO)
- **Social proof:** Others using it (FOMO)
- **Authority:** Expertise credibility (expert endorsement)
- **Liking:** Similarity to consumer (relatability)
- **Reciprocity:** Gift/value first, then ask (trial before buy)
- **Commitment:** Small yes leads to big yes (progressive engagement)

---

### NIVEL 5: IMPLEMENTACIÓN TÁCTICA

#### Brand Audit Checklist
- Logo consistency (all applications)
- Color palette adherence (Pantone matching)
- Typography standards (font files, weights)
- Voice consistency (style guide)
- Messaging alignment (core message in each asset)
- Visual library (complete asset inventory)
- Guidelines documentation (100+ page brand book)
- Competitive differentiation clarity (why different?)

#### Measurement Framework
- **Awareness:** Aided recall, unaided recall, top-of-mind
- **Perception:** Brand attributes, emotional associations
- **Loyalty:** Repeat purchase rate, NPS, customer lifetime value
- **Advocacy:** Word-of-mouth volume, referral rate, social mentions
- **Equity:** Price premium vs competitors, market share, brand valuation
